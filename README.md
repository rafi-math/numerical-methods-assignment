This repository contains clean and simple implementations of fundamental numerical methods used in computational mathematics.

The goal of this project is not just to make the code run — but to understand how classical numerical algorithms work step by step.

Everything here is written in plain Python, without heavy libraries, so the logic stays transparent.

📌 What This Repository Covers

🔎 Root Finding Methods

Bisection Method

Newton–Raphson Method

Regula Falsi Method

📈 Interpolation Methods

Newton Forward Interpolation

Newton Backward Interpolation

Divided Difference Interpolation

Interpolating Polynomial Construction

🧮 Linear System Solvers

LU Factorization

Gauss–Jordan Elimination

📊 Numerical Integration

Trapezoidal Rule (with error estimation)

📉 Ordinary Differential Equations

Euler’s Method

Runge–Kutta Method

🎯 Purpose of This Project

This project was built to:

Strengthen understanding of numerical analysis

Practice algorithm implementation from scratch

Build structured mathematical programming habits

Create a reusable reference for academic use

No shortcuts. No black-box libraries.
Just logic, mathematics and implementation.

🛠 Requirements

Python 3.x

No external libraries required
